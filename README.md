# TSP_Solver

## Prerequisites

- **Rust**  

## Usage
Cara untuk menjalankan program

1. Clone repository ini dengan perintah:
    ```bash
     https://github.com/henry204xx/TSP_Solver.git
    ```
2. Compile dan run program dengan
    ```bash
     rustc main.rs
    ```
    ```bash
     ./main
    ```
3. Contoh input matrix dapat dilihat pada file test
